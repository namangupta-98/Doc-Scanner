Document Scanner By Naman Gupta
===============================================================================================================

Requirements:
-python3.5+
-opencv-python 3.4.2.16
-opencv-contrib-python 3.4.2.16
-numpy
-scipy
-imutils
-matplotlib
================================================================================================================

1.Put the images you want to scan in "example_input" folder (checkout "example_input" for sample images).

2.Run scan.py file or if using python IDLE just double-click.

3.Select or Crop the image in the opened window and then CLOSE it.

3.The scanned images are saved in "example_output" folder. 

================================================================================================================

Avg. Time per Image:  0.8149330000000001
